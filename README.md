## Development

Preprequisites:

```bash
go install github.com/gzuidhof/tygo@latest # for generating API typings
go install github.com/mitranim/gow@latest # for watching for changes
```

Run server:

```bash
make run-server
```

Run client-only dev server:

```bash
cd ui && npm run serve:desktop
```
